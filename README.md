# Demonk Monk Simulation

An interactive AI-powered web application based on the classic Demon Monk river crossing puzzle. Built using Flask and Python with a BFS-based AI solver that generates optimal moves and hints.

## Features
- Interactive puzzle gameplay
- AI-based hint generation using BFS
- Constraint and invalid move validation
- Optimal solution comparison
- Responsive web interface

## Tech Stack
- Python
- Flask
- HTML/CSS
- BFS Algorithm
